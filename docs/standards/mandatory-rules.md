# Mandatory Rules

Estas regras nao devem ser quebradas por agentes.

## Regras obrigatorias

- Ler `README.md`, `AGENTS.md` e `docs/standards/index.md` antes de agir.
- Consultar apenas os arquivos tematicos necessarios para a tarefa.
- Nao versionar segredos.
- Nao duplicar regras ja documentadas em outro markdown tematico.
- Criar commit apenas ao fechar bloco funcional validado.
- Usar `Conventional Commits`.
- Registrar erro recorrente em `learned-lessons.md`.
- Registrar comando repetitivo seguro em `repetitive-commands.md`.
- Promover regra critica para este arquivo quando houver risco de repeticao, seguranca ou desperdicio.
- Minimizar uso de tokens, chamadas e leitura desnecessaria de arquivos.
