# Monitoring Standard

## Stack base

- `Prometheus`
- `Grafana`
- `Alertmanager`
- `node_exporter`
- `mysqld_exporter`

## Minimo esperado

- CPU
- RAM
- swap
- disco
- inode
- IO wait
- disponibilidade do Nginx
- disponibilidade do PHP-FPM
- latencia e conexoes do MariaDB

## Alertas minimos

- disco acima de 80%
- disco acima de 90%
- RAM sustentada alta
- swap em uso
- servico parado
- falha de backup
